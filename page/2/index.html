IP-guard 服务器安装前后需要有哪些注意事项？

发表于 2020-06-17 分类于 IP-GUARD Valine：

安装SQLserver时的注意事项：实例名称（如果客户本身的数据库有其他业务系统运行，需要注意，必要的话重新安装指定实例）；设置服务账户必须选择本地系统账户(Local System)。

参考文档：《SQL 2005 Express Installation.doc》

支持的Microsoft SQL Server数据库版本：

SQL Server 2000 SP4或以上
SQL Server 2005 SP1或以上
SQL Server 2008
SQL Server 2012
SQL Server 2016
SQL Server 2017
如果没有购买SQL Server并且客户规模较少，可以安装微软提供的免费的MSDE SP4、SQL Server Express，推荐2005 Express和2008 R2Express。 MSDE只支持2G的数据库容量，MSDE会受并发数量的影响，因此，MSDE一般只推荐用于测试，建议客户如需安装免费版本的数据库，应安装SQLServer Express版本。SQL server 2005 Express支持到4G的数据库大小。SQL server 2008 R2 Express 支持到10G的数据库大小。

为了保证服务器的正常工作，存放数据库文件的磁盘分区（数据库文件默认存放在服务器的安装目录下）最好为 NTFS 格式的，FAT32 格式的磁盘分区有单个文件大小不能超过 4G 的限制。

服务器安装成功后，建议先设置检验码再安装客户端，使服务器与客户端的连接更安全。

H3C华三无线用户VLAN内二层隔离

发表于 2020-06-14 更新于 2020-06-16 分类于 H3C Valine：

应用说明：

同一VLAN内，来自无线客户端的广播、组播报文会向所有放通该VLAN的AP上广播，而且在空间介质中广播报文通常使用最低速率进行发送。当广播报文比较多时，会占用较多的空口资源，在一定程度上影响到整个网络应用。
无线用户VLAN内二层隔离可以在AC上控制无线用户只能访问网关设备，而不能互相之间访问。这样可以大量减少整个WLAN网络的广播流量，提高WLAN网络的整体性能。
