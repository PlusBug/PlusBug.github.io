windows10下AMD显卡驱动无法安装的解决方法

发表于 2020-06-14 分类于 WINDOWS Valine：

具体参考：https://jingyan.baidu.com/article/7908e85cb3701aaf481ad2d9.html
问题：HD集成显卡带AMD显卡的笔记本在安装完系统以后无论使用官方下载的AMD显卡还是各种驱动软件均无法正常驱动。
解决：先把HD集成显卡更新基本适配器即可正常安装AMD独立显卡，然后再重新驱动HD显卡即可。

桌面图标删除后需要刷新才会消失

发表于 2020-06-14 分类于 WINDOWS Valine：

一般出现这个问题大多是系统没有这个设置，我们要在注册表添加一条一项。首先在运行输入regedit，然后依次打开HKEY_LOCAL_MACHINE/SYSTEM/CurrentControlSet/Control 在control上面右击选择新建 - 项，命名为UPData，然后在UPData上面在右击依然是新建 - 项，命名为UpDataModel。 然后注销下系统或者重新启动下Windows资源管理器

爱快可以远程桌面无法连接共享打印

发表于 2020-06-14 分类于 爱快 Valine：

场景：通过爱快路由器进行L2TP进行VPN对接，并且设置好相关路由表和ACL。两端内网可以互访通过远程桌面并且可以共享打印机。

问题：可以远程桌面，但是无法访问共享，造成共享打印无法打印。检查路由表及相关策略没有问题？

解决：断开L2TP连接，重新拨号认证即可。

安卓手机开机一直转圈圈

发表于 2020-06-14 分类于 Android Valine：

当安装了不当的软件APP造成冲突的情况下，开机的时候一直处于LOGO转圈圈的状态下无法正常开机。

解决方法：开机的时候反复按音量 - 即可进入安全模式状态开机然后删除造成冲突的APP 重启即可。
